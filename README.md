# 🚀 NeonQuiz AI — API Integrated Quiz Platform

NeonQuiz AI is a modern, interactive quiz application built using **HTML, CSS, and Vanilla JavaScript** with real-time question fetching from the Open Trivia API. The application features a futuristic neon UI, timer-based gameplay, scoring system, leaderboard, and persistent storage using localStorage.

This project demonstrates strong frontend fundamentals including state management, API integration, dynamic rendering, and UI architecture.

## ✨ Features

- 🌐 Real-time Question Fetching (Open Trivia API)
- ⏳ Countdown Timer per Question
- 🧠 Dynamic Score Tracking
- 🏆 Local Leaderboard System
- 💾 Persistent Data using LocalStorage
- 🎨 Neon Themed Modern UI
- 📱 Fully Responsive Design
- 🔄 Restart & Replay Functionality
- 🛡️ Secure Rendering (Prevents Injection Issues)

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Custom Styling + Animations)
- Vanilla JavaScript (ES6+)
- Open Trivia API
- LocalStorage API

No frameworks. No external libraries.
